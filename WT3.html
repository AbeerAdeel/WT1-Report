Benefits/LimitationsFunctionality

I wanted to gain exposure to using React Hooks which is the new way of implementing React code. I previously had experience with React but I learned and implemented React code using a class base structure. Throughout this whole co-op term I gained exposure to React Hooks as I implemented with React Hooks on many different tasks and features. I quickly learned that React Hooks is a much cleaner way of implementing React as it has much better functionality, better state management and is much more readable overall.
