Focus of Work Term

I want to focus my work term report on Angular which is a front end technology that I fell in love with across the work term. I primarily used React as my choice of front end technology but Angular is now my go-to. Below are aspects of Angular that makes it a great front end technology to use.

HTML and CSS Output

What you see on screen is purely generated from HTML and CSS which makes it easy to track where components are rendered on the screen in a browser inspect window.

Two-Way Data Binding

With using a two-way data binding method this makes sure that the model data changes automatically when any change is made.

TypeScript Capability

Angular uses TypeScript as its default to rendering components which is a much better alternative than JavaScript as you have a better idea of field types.

Easy Set Up

Angular is super easy to set up as you can easily generate new components as well as quickly start your web application much quicker than React.
