Focus of Work Term Report

I wanted to foucs my work term report about Machine Learning, Big Data and Artifical Intelligence which I learned in this work term. Below are steps and important processes that help Machine Learning the most powerful tool in Computer Science today. These tools and steps are used by prestigous companies such as Google, Facebook, Apple and many others!

DATA CLEANING

Data Cleaning is simply getting rid of data which isn't relevant in building the Machine Learning model.

DATA MINING

This process is simply discovering potential patterns in your large dataset which helps solidify the purpose of the Machine Learning model.

CLASSIFIERS

Using a classifier is the ideal method to train a Machine Learning model. Some examples of classifiers are Neural Networks, Regression, Forests, and Deep Learning.

MODEL TESTING

Testing models with different datasets and multiple classifiers will ensure high accuarcy and a better classifier score.
